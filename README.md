# Telecommunication Customer Churn Analysis

## Project Overview

This project analyzes customer churn in the telecommunications industry using Excel-based analytics and dashboarding techniques. The objective is to identify factors contributing to customer attrition and provide actionable business insights to improve customer retention.

The project includes KPI tracking, customer segmentation, churn analysis, and interactive dashboard visualizations.

---

## Business Objective

Customer churn directly impacts revenue and profitability. The goal of this analysis is to:

* Identify customers likely to churn
* Understand churn-driving factors
* Improve customer retention strategies
* Support data-driven business decisions

---

## Dataset Description

The dataset contains customer information such as:

* Customer ID
* Account Length
* State
* International Plan
* Voice Mail Plan
* Number of Customer Service Calls
* Day, Evening, Night Usage
* Charges and Revenue Metrics
* Churn Status

## Key Insights

* Customers with International Plans show higher churn rates.
* Frequent customer service complaints are strongly associated with churn.
* High-value customers require proactive retention strategies.
* Customer engagement directly influences retention.

---

## Recommendations

1. Improve customer support response times.
2. Create loyalty programs for high-value customers.
3. Monitor customers with frequent complaints.
4. Offer targeted retention campaigns.
